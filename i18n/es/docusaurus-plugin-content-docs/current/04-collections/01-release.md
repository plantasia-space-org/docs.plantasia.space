Publicar es el flujo único para ensamblar y lanzar una colección:

1. Elige los Audios, mundos y orbiters que pertenecen juntos.
2. Define el orden o las rutas que guían a lxs exploradores, y agrega notas sobre la experiencia buscada.
3. Confirma que todos los elementos estén publicados y accesibles; ajusta visibilidad y créditos de colaboración.
4. Publica para que lxs xPlorers puedan entrar y empezar a regenerar sonido dentro del recorrido curado.

Monitorea la participación y itera la colección conforme responde la comunidad.
