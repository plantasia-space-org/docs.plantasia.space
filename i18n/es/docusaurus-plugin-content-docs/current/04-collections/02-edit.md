---
title: Editar
---

Modifica una colección conforme recibes feedback:

- Reemplaza o reordena Audios para cambiar el arco emocional.
- Combina mundos y orbiters de forma distinta para abrir nuevas interacciones.
- Actualiza las notas o visuales de guía para que lxs exploradores se orienten.

Editar mantiene las colecciones vivas y ajustadas a cómo lxs xPlorers las recorren.
