---
title: Introducción
---

plantasia.space es una plataforma para música regenerativa: un espacio donde xPlorers lanzan sonidos, los colocan dentro de mundos entrelazados y los remodelan mediante exploración lúdica. Cada sonido se comparte para que pueda escucharse, recrearse y transformarse en nuevas formas.

Usa esta sección para entender cómo encaja la plataforma y cómo cada elemento creativo sostiene la música regenerativa.
