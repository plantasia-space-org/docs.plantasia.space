Un mapa simple de cómo se conectan los elementos creativos dentro de plantasia.space.

```
plantasia.space docs
│
├── plantasia.space
│   ├── Introducción
│   ├── Preguntas frecuentes
│   ├── Glosario
│   └── RX
│
├── Audios
│   ├── Resumen
│   ├── Publicar
│   ├── Editar
│   └── Usar
│
├── Mundos entrelazados
│   ├── Resumen
│   ├── Publicar
│   ├── Editar
│   └── Usar
│
├── Orbiters
│   ├── Resumen
│   ├── Publicar
│   ├── Editar
│   └── Usar
│
└── Colecciones
    ├── Resumen
    ├── Publicar
    ├── Editar
    └── Usar
```

Usa la navegación lateral para saltar a cualquier parte de este árbol.
