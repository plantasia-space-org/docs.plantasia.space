title: Diagrama
---

Un mapa simple de cómo se conectan los elementos creativos dentro de plantasia.space.

```
plantasia.space docs
│
├── plantasia.space
│   ├── Introducción
│   ├── Preguntas frecuentes
│   ├── Glosario
│   └── RX
│
├── Audios
│   ├── Panorama
│   ├── Publicar
│   ├── Editar
│   └── Usar
│
├── Mundos entrelazados
│   ├── Panorama
│   ├── Publicar
│   ├── Editar
│   └── Usar
│
├── Orbiters
│   ├── Panorama
│   ├── Publicar
│   ├── Editar
│   └── Usar
│
└── Colecciones
    ├── Panorama
    ├── Publicar
    ├── Editar
    └── Usar
```

Usa la navegación lateral para saltar a cualquier parte de este árbol.
